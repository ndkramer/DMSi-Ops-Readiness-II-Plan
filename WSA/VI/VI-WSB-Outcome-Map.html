What Changes from the Stage-Gate Model

Parallel Execution Unlocked

VI-OC-06 (Log Migration), VI-OC-07 (Synthetic Monitoring), VI-OC-04 (Host Instrumentation), and VI-OC-16 (PagerDuty AIOps) all start after VI-OC-03 and run concurrently. PagerDuty AIOps is already underway in parallel. Dynatrace is currently in a vendor POC phase (before the dated full rollout on the timeline).

Iterative Loops Made Explicit

VI-OC-05 and VI-OC-09 have 30-day validation windows with max 2 iterations. If validation fails, the team loops back within the same outcome instead of "re-entering a previous stage." Fallback triggers VI-OC-15.

Conditional Work Clarified

A2W monitoring (VI-OC-13), advanced capabilities (VI-OC-14), and legacy fallback (VI-OC-15) are planned but not scheduled. They activate only when trigger conditions are met, eliminating phantom blockers.
