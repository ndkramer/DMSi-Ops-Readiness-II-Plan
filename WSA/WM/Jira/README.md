# WM — Jira exports

## Dated exports

- **`WM-Jira-MM-DD-YYYY-json.json`** — saved Jira API exports. **Retain multiple** dated files for history (same pattern as **`WSA/PA/Jira/PA-Jira-*.json`**).

## Target-state import JSON

- **`../Output/{filePrefix}-WBS-Jira-Import.json`** (default **`WM-…`**) — generated by **`node Scripts/wm-wsb-to-jira-import.js`** (paths / prefix from **`dynamo-os.config.cjs`** when **`Scripts/planning-path-context.js`** resolves the toolkit); uploaded with **`dynamo-plan jira import-wm`** (or **`node Scripts/jira-import-wm.js`**).
- **`../Output/Archive/WM-WBS-Jira-Import-MM-DD-YYYY.json`** — snapshots from `node Scripts/wbs-load-prep.js WM`.
