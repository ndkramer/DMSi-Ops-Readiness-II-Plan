# WM — Jira exports

- **`WM-Jira-MM-DD-YYYY-json.json`** — saved Jira API exports. **Retain multiple** dated files for history (same pattern as **`WSA/PA/Jira/PA-Jira-*.json`**).

## Target-state import JSON

- **`../Output/WM-WBS-Jira-Import.json`** — generated by `node Scripts/wm-wsb-to-jira-import.js`; used by `Scripts/jira-import-wm.js` for upload workflows.
- **`../Output/Archive/WM-WBS-Jira-Import-MM-DD-YYYY.json`** — snapshots from `node Scripts/wbs-load-prep.js WM`.
